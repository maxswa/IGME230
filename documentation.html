Documentation

I started with a simple board made of straight rectangular divs. I later made them transparent against a hand-drawn board made by MJ. I skewed them to help the illusion of perspective.

I used some of the click and drag code from the magnetic words assignment.

In order to get the undo button to work, I had to clone the board to create an "old" version. I had to write a custom function to do this, since the board array is full of objects that represent the points on the board. When cloning arrays, objects are copied by reference, when I needed them to be passed by value.
